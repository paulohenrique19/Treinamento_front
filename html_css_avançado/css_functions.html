CSS Functions

Podemos aplicar funções no CSS, a sintáxe é: (função: + ação ou ação(parâmetro))

Transform

Aplicamos transformações no elemento

transform: translate(desloc X, desloc Y), aplicamos um deslocamento do elemento, semelhante ao position

transform: rotate(), rotaciona o elemento em alguns graus

transform: scale(), aumenta o tamanho do elemento

Aplicando transform: translate(100px, 200px) rotate(40deg) scale(1); no elemento

1

Math functions

Aplicamos funções matemáticas nos atributos dos elementos

calc(), podemos fazer um cálculo de 1, 2 ou mais valores usando as 4 operações básicas

max(), selecionamos o maior valor entre dois elementos

min(), selecionamos o menor valor entre dois elementos

clamp(), a página escolhe um valor de 3 em que se adequa melhor ao elemento

Aplicando no elemento: width: calc(30px + 1rem); height: max(calc(20% + 10vh), min(30vh, 100px)); font-size: clamp(1rem , 7vw, 4rem);

1

Podemos usar outras funções matemáticas como:

Funções trigonométricas: sin(), cos(), tan()

Funções exponenciais: pow(), sqrt()

Filter functions

Podemos aplicar filtros de estilização nos elementos

filter: blur, aplica o efeito blur no elemento

filter: brightness, aplica um brilho no elemento

filter: contrast, aplica um efeito de contraste na imagem

filter: drop-shadow, aplica uma sombra no elemento

filter: opacity, aplica uma transparência no elemento

filter: invert, inverte as cores do elemento

1

filter: hue-rotate(), troca a cor do elemento a partir de um deslocamento partindo da cor atual

Aplicando hue-rotate(100) no elemento com background-color: aqua

Color functions

Aplicando cores no elemento

Funções:

rgb(x, y, z) (0-255, 0-255, 0-255)

hsla(x, y, z, opacidade) (0-300, 0-100%, 0-100%, 0-1)

color-mix(in + tipo X, cor1 tipo X, cor2 tipo X), mistura cores

background-color: color-mix( in hsl, hsl(), hsl() )

Gradient functions

Aplica um efeito "degradê" no background

as gradient functions são vistas como imagens pelo CSS, então podemos aplicá-las também usando background-image:

aplicando: background: linear-gradient( 90deg, red, green, aqua, blue );

aplicando: background: linear-gradient( to left, red, green, aqua, blue );

aplicando: background: linear-gradient( to left, red, green 80%, aqua, blue );

Aplicando radial-gradient

aplicando: background: radial-gradient(blue 50%, red);

Clip Path

Podemos aplicar formas específicas aos elementos

As principais são: clip-path: circle | clip-path: polygon

clip-path: circle: Aplica a forma de um círculo

clip-path: polygon: Aplica uma forma específica

aplicando: clip-path: polygon(50% 0%, 0% 100%, 100% 100%);
