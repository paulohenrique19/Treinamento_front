HTML Forms

Serve para capturar dados

Button, atributos:

type

Por padrão, um botão é do tipo submit quando estiver dentro de um form

Botão com type submit:

reset: reseta o input

button: não envia o formulário, mas faz alguma ação

autofocus: coloca uma marcação no botão depois de apertá-lo

disabled: desativa o botão

name: atribui um nome ao botao

value: atribui um valor ao botao para ser enviado ao formulario

Input, atributos:

Input Text

name: o valor do input ao enviar o formulário

type: define o tipo de dado do input

atributos gerais são atributos em que muitos deles poderão passar por vários elementos

value: coloca um valor por padrão no input

autocomplete: tenta completar o elemento dentro de acordo com o que está sendo inserido

size: altera o tamanho do input

autofocus: coloca um destaque no input após alguma ação

disabled: desabilita o botão

readonly: permite apenas leitura do campo e não permite alterações

form: serve para linkar algum formulario a algum input

required: requere que o campo seja preenchido antes de submeter

placeholder: coloca um texto no fundo do campo antes dele ser preenchido

Input Number

type: number -> aceita apenas números

max & min: define um limite númerico que o input aceita

step: pula os números disponíveis, ex, se o step for 2 e temos números de 0 a 10, isso quer dizer que ele só irá aceitar números pares de 0 a 10

Input Email

Por padrão, inputs do tipo email requerem que pelomenos um @ seja colocado

multiple: permite que multiplos emails sejam enviados no formulário separados por vírgula

minlenght & maxlength: define um limite para o tamanho do e-mail

pattern: expressão regular para validar o campo, ex se eu colocar pattern=".+gmail\.com", ele vai aceitar somente emails que terminam com @gmail.com

title: mensagem de "erro" no input

Input Password

Por padrão, esconde o que está sendo digitado

"inputmode: numeric" serve para abrir um teclado númerico de digitação de senhas (apenas para celulares)

podemos usar o pattern também para forçar o usuário a criar uma senha mais segura

Input File

Serve para enviar arquivos

No form, precisamos colocar o atributo e valor: enctype="multipart/form-data"

multiple: serve para enviar vários arquivos

accept: serve para limitar os tipos de arquivos que o formulário aceita

Input Range

Controle para selecionar valor numérico

step: serve para definir valores a cada número de passos definidos, ex: step: 20, pula de 20 em 20

Input Color

Serve para enviar cores no formulário

podemos usar o datalist para pré-definir uma lista de cores linkando um id da lista ao input com o atributo list

Input Checkbox

Input por caixas de marcação

checked: deixa uma marcação em algum campo por padrão

Input Radio

Semelhante ao checkbox

Projetado para selecionar uma única opção de um grupo de opções

Se trocarmos o valor do atributo name, podemos selecionar mais opções

Checked funciona aqui também

Input Hidden

Esconde um input, mas pode enviar dados

Label

Serve para nomear um campo, quando clicamos na label, o campo gera uma marcação

for: nomeia um campo

Text area

Cria uma área de texto

Podemos definir tamanho com as propriedades rows e cols e também permitir wrap

Podemos inserir também as propriedades placeholderm disbled e required

Select

Serve para selecionar elementos de uma lista, ela pode ser agrupada com o optgroup ou não

multiple: permite que multiplos valores sejam selecionados

size: determina um tamanho

Fieldset

Agrupamento de campos
