CSS Grid: Introdução

Toda grid é composta de 2 principais grupos:

'container: o pai' e 'itens: o(s) filhos'

Container (pai)

-display: grid

-grid-template:

  • grid-template-columns;
  • grid-template-rows;
  • grid-template-areas;

Itens (filhos)

-grid-column

  • -grid-column-start;
  • -grid-column-end;

-grid-row;

  • -grid-row-start;
  • -grid-row-end;

grid-template-columns

Aumenta o tamanho da largura??

Separamos o conteúdo em "caixas" alinhadas e definimos o espaço ocupado por cada um deles

Aqui estamos definindo grid-template-columns: 1fr 1fr 1fr; logo, cada um dos 3 itens terá uma fração da linha ocupada

Aplicando tamanhos diferentes para os itens:

grid-template-columns: 200px 100px 50px;

*não é certeiro:* Se aplicassemos somente grid-template-columns: 200px; o display dos itens seria block, então temos que aplicar para todos

Aplicando repetição de itens na linha:

grid-template-columns: repeat(3, 1fr) | primeiro escopo: nº de elementos por linha (3), segundo escopo: tamanho dos elementos (1fr)

O que acontece se aplicar um número maior de repetição para um número menor disponível de elementos?

Aplicando grid-template-columns: repeat(6, 1fr) para 3 itens

Como temos 3 itens e aplicamos 6 repetições, os itens ocupam metade da linha (3/6) = 50%, porque o CSS espera que colocamos mais 3 itens para ocupá-la inteira

grid-template-rows

Aumenta o tamanho da linha dos elementos da grid

Aplicando: grid-template-rows: 280px 1fr 2fr 50%;

O primeiro elemento terá 280px de altura, o segundo e o terceiro 1fr e 2fr respectivamente e o último ocupará 50% do tamanho disponível

Combinando grid-template-columns com grid-template-rows

Aplicando grid-template-columns: repeat(2, 1fr);

Aplicando grid-template-rows: 200px 100px;

1º elemento com 200px de altura e 1fr de largura
2º elemento com 200px de altura e 1fr de largura 3º elemento com 100px de altura e 1 fr de largura 4º elemento com 100px de altura e 1 fr de largura

Grid template areas

Podemos separar os elementos dentro do container definindo uma grid dividida em literais áreas

Style:

#grid-template-areas { display: grid; grid-template-areas: "A B B" "A B B" "A C D"; height: 50vh; border: solid green; }

OBS: Podemos definir qualquer nome para as áreas

Como já definimos as áreas, agora é só inserir os elementos nas áreas A e B somente pois o resto se autocompleta

Ajustando A: #grid-template-areas div:nth-child(1) { grid-area: A; }

Ajustando B: #grid-template-areas span:nth-child(2) { grid-area: B; }

Propiedades de alinhamento para grid

Existem 9 propriedades fundamentais

6 aplicadas em container

Content:

  • 'align-content'
  • 'justify-content'
  • 'place-content'

Items:

  • 'align-items'
  • 'justify-items'
  • 'place-items'

3 aplicadas em item

Self:

  • 'align-self'
  • 'justify-self'
  • 'place-self'

Podemos separar em 3 grupos: 'align', 'justify', 'place'

E cada um deles irá observar ou o

  • conteúdo do elemento 'content'
  • itens do elemento 'items'
  • o próprio elemento 'self'
