Sizing: Flex Basis, Flex Grow e Flex Shrink

Flex Basis

Tamanho do item relativo ao eixo principal da disposição de itens, se o eixo for invertido, os itens serão dispostos de forma invertida

Pode assumir altura ou largura, tudo depende da disposição dos itens

Flex Shrink

Uma propriedade que permite o encolhimento automático dos itens vistos no Flex Basis e Flex Grow

Revisão e propriedades extras

Propriedades que aplicamos no container

  • display
  • flex-flow: shorthand para os de baixo
    • flex-direction
    • flex-wrap
  • gap: shorthand para row-gap e column-gap
    • row-gap: espaço entre linhas
    • column-gap: espaço entre colunas
  • justify-content
  • align-items
  • align-content

Propriedades que aplicamos no child

  • flex
  • flex-grow
  • flex-shrink
  • flex-basis
  • align-self: útil para alinhar elementos (flex-start,end,flex-end)
